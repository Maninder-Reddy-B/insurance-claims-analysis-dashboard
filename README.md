# Insurance Claims Analysis Dashboard

Interactive Insurance Claims Analytics Dashboard built using HTML, CSS, JavaScript, Excel, and DAX concepts.

### Features

* Claims KPI tracking
* Approval rate analysis
* Rejection analysis
* Interactive filters
* Trend visualizations
* Export cleaned data

### Technologies

* HTML
* CSS
* JavaScript
* Excel
* DAX
